CentOS에서 Python 업데이트 시 이슈 (Feat. PostgreSQL)

2024-12-19 태더

개요

최근 Python 3.6 기반의 레거시 서비스를 Python 3.9로 업데이트하였습니다. 주요 라이브러리들을 전부 업데이트하였고, 그 중 PostgreSQL 연결시 사용하는 psycopg2-binary 패키지가 업데이트 되면서, 서버 내 PostgreSQL Client 구성이 필요하게 되었습니다. 이 포스팅은 PostgreSQL Client의 구성과 psycopg2-binary 패키지 업데이트 이슈를 해결하는 과정을 정리합니다.
